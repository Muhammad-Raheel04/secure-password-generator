Secure Password Generator

A secure and user-friendly password generator that helps users create strong and customizable passwords. The application also integrates a custom AI chatbot powered by Chatbase to assist users in real time.

Overview

This project is a client-side web application designed to generate secure passwords based on user-selected criteria. It focuses on simplicity, security, and usability, while also providing interactive guidance through an integrated chatbot.

Features

Generates strong and secure passwords
Customizable password length
Option to include uppercase letters
Option to include lowercase letters
Option to include numbers
Option to include special characters
Copy password to clipboard
Responsive and modern user interface
Integrated Chatbase chatbot for user assistance

Technologies Used

HTML5
CSS3
JavaScript (ES6)
Chatbase.co API

How to Use

Open the application in a web browser

Select the desired password length and options

Click the Generate Password button

Copy the generated password

Use the chatbot if guidance is required

Security Considerations

Passwords are generated locally in the browser. No password data is stored, logged, or transmitted, ensuring user privacy and security.
